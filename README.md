# Smart Parking System with AI

AI-based parking space detection using Python and OpenCV.

## Features
- Real-time parking slot detection
- Image processing using OpenCV
- Automated space monitoring — no manual effort needed

## Technologies Used
- Python
- OpenCV
- VS Code

## How to Run
1. Install dependencies: `pip install opencv-python cvzone`
2. Run `Picker.py` to mark parking slots
3. Run `main.py` to start detection
